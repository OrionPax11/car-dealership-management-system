## 🚀 Features Included in the Full Source Code

> **⚙️ Technical Note:** 
> *This is a source code template designed to give developers a massive head start. Features like the Checkout/Payment module and City Selection are **Simulated UI Flows**. They are beautifully designed and track data in the database, but they are not wired to a live banking API (like Stripe) or a live map API. They are built perfectly for you to easily wire up to your preferred APIs!*

### 👤 Customer Experience
*   💎 **Glassmorphism UI:** A stunning, modern, and highly responsive aesthetic.
*   🔍 **Live Inventory Catalog:** Advanced searching and dynamic category filtering.
*   💳 **Simulated Checkout:** Complete purchase flow with color selection and dynamic delivery tracking.
*   📦 **Customer Dashboard:** Track orders in real-time (Confirmed → In Transit → Delivered).
*   📅 **Test Drive Booking:** Integrated scheduling system.
*   📄 **Dynamic PDF Invoices:** Auto-generated, downloadable receipts for every purchase.

### 🏢 Staff Dashboards (Role-Based Access)
*   👑 **Super Admin Dashboard:** Manage all inventory, global users, and system settings.
*   📋 **Inventory Manager Portal:** Add, edit, upload images, and delete vehicle listings in real-time.
*   🚚 **Delivery Manager Portal:** A beautiful drag-and-drop Kanban board to manage shipping statuses.
*   📧 **Support Inbox:** Centralized help center for staff to reply to customer inquiries.

### 🛡️ Security & Tech Stack
*   **Backend:** Node.js, Express.js
*   **Database:** PostgreSQL (Sequelize ORM)
*   **Frontend:** Pure Vanilla JavaScript, HTML5, Custom CSS (No bloated frameworks!)
*   **Security:** bcrypt Password Hashing, Session-based Auth, Helmet Headers, Rate Limiting.
